# 发布指南

## 发布准备

1. 确保已经登录 npm 账户

```bash
npm login
```

2. 验证当前 npm 用户

```bash
npm whoami
```

3. 执行发布命令

```bash
npm publish
```

## 更新版本

如需更新版本，修改 package.json 中的版本号并重新发布：

1. 升级补丁版本（修复 bug）

```bash
npm version patch
```

2. 升级小版本（新增功能）

```bash
npm version minor
```

3. 升级主版本（不兼容变更）

```bash
npm version major
```

## 测试全局安装

发布后，可以通过以下命令测试全局安装：

```bash
npm install -g ai-code-review
ai-code-review
```
